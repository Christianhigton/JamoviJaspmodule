# Disabled Analyses

These files are retained for future development but are intentionally excluded from the current jamovi build.

Current active analyses:

- JASP Enhanced ANOVA
- JASP Enhanced Repeated Measures ANOVA

Disabled analyses:

- ANCOVA
- MANOVA
- Bayesian ANOVA
- Bayesian Repeated Measures ANOVA
- Bayesian ANCOVA

To re-enable an analysis, move its jamovi yaml files back to `../jamovi/` and its R implementation file back to `../R/`, then rebuild the module.
