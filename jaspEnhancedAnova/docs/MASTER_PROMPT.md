# Master Prompt Addition

Feature parity is mandatory. The ANOVA module must reproduce every option available in the current JASP ANOVA module, including descriptive statistics, effect sizes, Welch and Brown-Forsythe corrections, custom contrasts, Bayesian informed hypotheses, post hoc procedures, descriptive plots, raincloud plots, marginal means, simple effects, nonparametric tests, and export functions. Any missing feature must be documented and justified. The resulting module should exceed standard jamovi ANOVA functionality while retaining a native jamovi user experience.

The Repeated Measures ANOVA module must reproduce every option available in the JASP Repeated Measures ANOVA module, including repeated measures factor construction, model builder, sphericity testing, Greenhouse-Geisser and Huynh-Feldt corrections, contrasts, Bayesian informed hypotheses, post hoc testing, descriptive plots, bar plots, raincloud plots, marginal means, simple main effects, nonparametric repeated measures procedures, and export functionality. Missing features must be documented and justified. The final module should exceed the capabilities of the standard jamovi Repeated Measures ANOVA while retaining a native jamovi user experience.
