JASP Enhanced ANOVA for jamovi

A focused jamovi module exposing two active analyses: Classical ANOVA and Repeated Measures ANOVA.

Development build. This project is currently scoped to two working analyses. Other JASP ANOVA-family analyses are retained in the repository under jaspEnhancedAnova/disabled/ and can be re-enabled later.

Active Analyses

JASP Enhanced ANOVA

Classical fixed-effects ANOVA with descriptives, effect sizes, diagnostics, custom contrasts, post hoc tests, marginal means, simple effects, raincloud-style plots, and generated residual/prediction columns.

JASP Enhanced Repeated Measures ANOVA

Wide-format repeated-measures ANOVA using selected cells as one within-subject factor, with repeated-measures contrasts, plots, post hoc tests, Friedman fallback, and subject-level output columns.

Install

Build artifact:

jaspEnhancedAnova_0.1.0.jmo

For local manual install on this machine:

unzip -q jaspEnhancedAnova_0.1.0.jmo -d /private/tmp/jaspEnhancedAnova-install
ditto /private/tmp/jaspEnhancedAnova-install/jaspEnhancedAnova \
  "$HOME/Library/Application Support/jamovi/modules/jaspEnhancedAnova"

Fully quit and reopen jamovi after installing.

Build

cd jaspEnhancedAnova
'/Library/Frameworks/R.framework/Versions/4.4-arm64/Resources/library/node/node-darwin/bin/node' \
  '/Library/Frameworks/R.framework/Versions/4.4-arm64/Resources/library/jmvtools/node_modules/jamovi-compiler/index.js' \
  --build '.' \
  --home '/Applications/jamovi.app' \
  --assume-app-version '2.7.0' \
  --jmo '../jaspEnhancedAnova_0.1.0.jmo'
