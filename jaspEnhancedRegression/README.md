# JASP Enhanced Regression for jamovi

This is an independent jamovi module scaffold for adapting analytical functionality from the open-source JASP Regression module into a jamovi-native workflow.

Design principle: JASP functionality. jamovi usability.

Current status: scaffold only. The analysis entries, option surfaces, feature inventories, attribution, and placeholder outputs are in place. Statistical implementations still need to be ported and verified against JASP before release.

Upstream reference: https://github.com/jasp-stats/jaspRegression

License: GPL (>= 2), matching the upstream JASP module license declared in its DESCRIPTION file.

## Included Analysis Shells

- JASP Enhanced Correlation
- JASP Enhanced Linear Regression
- JASP Enhanced Logistic Regression
- JASP Enhanced Generalized Linear Model
- JASP Enhanced Bayesian Correlation
- JASP Enhanced Bayesian Linear Regression
- JASP Enhanced Bayesian Logistic Regression
